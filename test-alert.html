🔔 告警弹窗功能已实现

✅ 功能说明

位置：弹窗固定显示在页面右上角
动画效果：
  • 从右侧滑入的入场动画
  • 呼吸灯脉动效果，提醒用户有告警
  • 平滑的关闭动画
交互功能：
  • 点击右上角 × 按钮可以关闭弹窗
  • 当有新的告警产生时，弹窗会自动重新显示
  • 当所有告警都恢复正常时，弹窗会自动关闭
样式：
  • 半透明玻璃态效果，与整体设计风格统一
  • 红色系主题，突出告警的重要性
  • 支持多条告警信息同时显示

🚀 测试方法

1. 访问开发服务器：http://localhost:5173/

2. 调整阈值设置到较低的值，触发告警：

  • CPU 阈值调低到当前 CPU 使用率以下
  • 内存阈值调低到当前内存使用率以下
  • GPU 阈值调低到当前 GPU 负载以下

3. 观察右上角是否出现告警弹窗

4. 测试关闭按钮功能

5. 调整阈值恢复正常，观察弹窗是否自动消失

  • 新增 AlertPopup.vue 组件：独立的告警弹窗组件
  • 修改 App.vue：
    • 导入 AlertPopup 组件
    • 添加告警显示状态管理
    • 添加告警变化监听，自动显示弹窗
    • 替换原有的顶部告警条为右上角弹窗
  • 修改 style.css：移除原有的 alert-bar 相关样式
