# 监控系统功能更新说明

## 📅 更新时间
2026-03-12

## 🔧 本次更新（部署与龙虾接入）

### 1. WebSocket 通道拆分，避免角色混用冲突
- 新增 `WS /ws/stream`：前端订阅实时监控流
- 新增 `WS /ws/report`：龙虾节点上报心跳与负载
- 保留 `WS /ws` 作为兼容通道（历史客户端可继续使用）
- 前端默认连接已切到 `/ws/stream`

### 2. 增加龙虾心跳实时可视化
- 后端新增 `GET /api/heartbeats`
- 实时 payload 增加 `heartbeats` 字段（每秒推送）
- 首页“管理节点”区域新增“实时心跳”列表，展示：
  - 在线状态（online/stale/never）
  - 距离上次心跳秒数
  - 最近上报时间

### 3. 首页接入教程升级（可直接转发给龙虾同事）
- 新增服务器 `IP/域名` 与端口输入
- 自动生成可直接运行的 Python 上报脚本模板
- 自动拼装目标上报地址：`ws(s)://<host>:<port>/ws/report`

### 4. 服务器部署连接策略优化
- 前端默认 API 地址改为：`http(s)://当前页面域名:8000`
- 前端默认 WS 地址改为：`ws(s)://当前页面域名:8000/ws/stream`
- 降低“部署后还在访问 127.0.0.1 导致 404”风险

## 🔧 功能调整列表

### 1. 告警展示方式优化：顶部告警条 → 右上角弹窗
#### 🔍 原实现
- 告警信息固定显示在页面顶部的 alert-bar 中
- 占用页面头部空间，影响主内容展示
- 没有关闭功能，告警期间一直显示

#### ✨ 新实现
- **位置**：弹窗固定显示在页面右上角（position: fixed）
- **层级**：z-index: 1000，确保在所有元素最上层
- **动画效果**：
  - 从右侧平滑滑入（transform: translateX）
  - 红色呼吸灯脉动效果（animation: pulse）
  - 平滑的关闭过渡动画
- **交互功能**：
  - 点击右上角 × 按钮可以手动关闭弹窗
  - 新告警产生时自动重新显示
  - 所有告警消除时自动关闭
- **样式设计**：
  - 半透明玻璃态效果，与整体设计统一
  - 红色系主题，突出告警重要性
  - 支持多条告警信息同时显示，每条独立分隔

#### 📝 修改文件
- 新增：`/frontend/src/components/AlertPopup.vue` - 独立告警弹窗组件
- 修改：`/frontend/src/App.vue` - 集成弹窗组件，替换原有告警条
- 修改：`/frontend/src/style.css` - 移除原 alert-bar 样式，新增弹窗相关样式

---

### 2. 阈值设置持久化功能
#### 🔍 原实现
- 阈值设置仅保存在内存中
- 刷新页面后恢复默认值
- 没有保存功能，每次打开页面都需要重新调整

#### ✨ 新实现
- **存储方式**：使用 localStorage 本地存储
- **UI 优化**：
  - 阈值设置区域新增"保存设置"按钮（绿色主题）
  - 按钮与标题水平布局，节省空间
- **功能逻辑**：
  - 页面加载时自动读取保存的阈值
  - 点击保存按钮持久化当前设置
  - 保存成功后弹出提示"阈值设置已保存"
  - 错误处理：JSON 解析失败时使用默认值
- **默认阈值**：
  - CPU: 80%
  - 内存: 80%
  - GPU: 85%
  - 节点CPU: 85%

#### 📝 修改文件
- 修改：`/frontend/src/App.vue`
  - 添加 `loadThresholds()` 加载函数
  - 添加 `saveThresholds()` 保存函数
  - 阈值设置区域 UI 重构
- 修改：`/frontend/src/style.css`
  - 新增 `.panel-header` 样式
  - 新增 `.save-btn` 按钮样式
